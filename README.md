# Аренда техники и мебели — Куйньон

Микробизнес по аренде мониторов, столов, стульев, освещения и колонок в Куйньоне.
Коллаб с Юрой. Договор аренды через жену-вьетнамку.

## Концепция

Модель **just-in-time inventory** — покупаем предмет только после подтверждённой заявки и предоплаты.
Нулевые стартовые вложения, риск минимален. После 2–4 аренд предмет окупается и остаётся в парке.

## Ассортимент

- Мониторы (27", 32", возможно 24")
- Компьютерные стулья (базовые и эргономичные)
- Столы (простые и электрические стоячие)
- Колонки
- Осветительные приборы

## Сайт

Файл `index.html` — статический каталог на GitHub Pages.
Конфигурация инвентаря — `catalog.json` (правишь только его, сайт перестраивается сам).

**GitHub Pages:** Settings → Pages → Source: `main`, folder: `/ (root)`

## Целевые клиенты

1. **Организаторы мероприятий** — конференции, корпоративы, свадьбы (разовые, но крупные заказы)
2. **Местные офисы** — временное расширение, онбординг сотрудников
3. **Фотографы / видеографы** — второй монитор для ретуши и монтажа
4. **Кафе и коворкинги** — оптовая аренда на месяц

## Процесс работы

1. Клиент оставляет заявку через сайт или Telegram
2. Берём предоплату 100% (скриншот переписки = договор)
3. Покупаем в магазине (Thế Giới Di Động, Điện Máy Xanh)
4. Доставляем → сдаём → после аренды забираем
5. После 2–4 аренд предмет переходит в постоянный парк

## Партнёрство

**Юра** — доставка и хранение. Снижает операционку, ускоряет выход на окупаемость.
